Warm-up

  • Jogging x5 field rounds + gun + mask
  • Hold each stretch for 2s
  • Active standing adductors (x5 each side)
  • Active 90/90 hamstring stretch (x5 each side)
  • Active supine bicep femoris stretch (x5 each side)
  • Active kneeling hip flexor (x5 each side)
  • Inchworm to worlds greatest stretch (x5 each side)
  • Down dog dive bomb (x5 each side)
  • Crab (x5 each side)
  • Bonus: You can finish with some 1v1s before diving into the main session
Active Stretching
Dynamic Stretching

Snake Fatigue Drill

  • Obstacle + 1 target
  • 20s mountain climbers - Snake Crawl - Quick pop and shoot
  • 3 attempts to hit target with 1-2 balls
  • 3 reps each side x3 sets
  • Rest 2 minutes per set

Snake Fatigue Drill Progression

  • Obstacle + 1 target
  • Breakout Sprint - Dive 'n' Crawl - Quick pop and shoot
  • 1 attempt to hit target with 3-4 balls
  • 3 reps each side x3 sets
  • Rest 2 minutes per set

Snake Scenario

  • Obstacles + 5-6 targets
  • 2 interior and 4 snake possible shots
  • 1 interior and snake bunker calls per rep
  • 3 reps each side x3 sets
  • Rest 2 minutes per set
  • Delta call: Dorito side back bunker usually close to Dorito 1
  • Echo call: Usually the 2nd back bunker from the Snake Side

Cooldown

  • Hold each stretch for 30s each side
  • Long Standing Calf Stretch
  • Short Standing Calf Stretch
  • Standing adductor stretch
  • 90/90 hamstring stretch
  • Supine bicep femoris stretch
  • Kneeling hip flexor stretch
  • Supine figure 4
