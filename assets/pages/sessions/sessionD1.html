Sliding Session

Beginner

Overview

Focus on your sliding technique, timing, running and confidence into getting to bunkers safely and without hits. In this workout, we will first focus on understanding the technique behind a solid sliding form, once that is understood, we then move forward to practicing this technique and increasing the intensity gradually.

Warm-up

  • Jogging x5 field rounds + gun + mask
  • Hold each stretch for 2s
  • Active standing adductors (x5 each side)
  • Active 90/90 hamstring stretch (x5 each side)
  • Active supine bicep femoris stretch (x5 each side)
  • Active kneeling hip flexor (x5 each side)
  • Inchworm to worlds greatest stretch (x5 each side)
  • Down dog dive bomb (x5 each side)
  • Crab (x5 each side)
  • Bonus: You can finish with some 1v1s before diving into the main session

Sliding Drill 1

  • 7 markers in a zig-zag pattern
  • Alternatively: 2 markers (6 total slides)
  • 2 rounds of 6 slides x2 sets
  • Rest for 1 minute between each round
  • Rest for 2 minute between each set

Sliding Drill 2

  • 4 markers & 2 sticks/ yarn
  • 15 meters distance run
  • 0.5 meters high
  • 6 reps x2 sets with full gear
  • Rest for 2 minute between each set

Sliding Drill 3

  • 3 markers & 3 targets
  • 9 meter distance from each target
  • 6 meter distance run from the middle
  • 3 shots per target
  • 1 rep x2 sets

Cooldown

  • Hold each stretch for 30s each side
  • Long Standing Calf Stretch
  • Short Standing Calf Stretch
  • Standing adductor stretch
  • 90/90 hamstring stretch
  • Supine bicep femoris stretch
  • Kneeling hip flexor stretch
  • Supine figure 4
