Warm-up

  • Jogging x5 field rounds + gun + mask
  • Hold each stretch for 2s
  • Active standing adductors (x5 each side)
  • Active 90/90 hamstring stretch (x5 each side)
  • Active supine bicep femoris stretch (x5 each side)
  • Active kneeling hip flexor (x5 each side)
  • Inchworm to worlds greatest stretch (x5 each side)
  • Down dog dive bomb (x5 each side)
  • Crab (x5 each side)
  • Bonus: You can finish with some 1v1s before diving into the main session

Dynamic Dorito Moves

  • 1st video is theory, 2nd video is drill
  • Obstacles + targets
  • Targets are to be placed on the mirror and were one can wrap
  • 5 reps each side x3 sets
  • 30 second rest between reps
  • 1 minute rest between sets
  • Tip: The basic idea of this drill is to take out your mirror, wrap & go

Tactical Decision Making

  • 1v1 or 2v2 + loader
  • Option 1: Gameplay is done only on the dorito side
  • Option 2: One player/team on dorito side, other player/team plays in the middle/snake
  • 5 reps each side x3 sets
  • 1 minute rest between reps
  • 2 minutes rest between sets

Game Simulation (progression)

  • 1v1 or 2v2 + loader
  • Guns are placed in the middle of the field
  • Players start at the starting station, run to the gun and back, game starts
  • Option 1: Gameplay is done only on the dorito side
  • Option 2: One player/team on dorito side, other player/team plays in the middle/snake
  • 5 reps x3 sets
  • 1 minute rest between reps
  • 2 minutes rest between sets

Cooldown

  • Hold each stretch for 30s each side
  • Long Standing Calf Stretch
  • Short Standing Calf Stretch
  • Standing adductor stretch
  • 90/90 hamstring stretch
  • Supine bicep femoris stretch
  • Kneeling hip flexor stretch
  • Supine figure 4
